# SplitBot

Telegram бот для разбивки счёта.

## Установка на VPS

```bash
# Клонируй / скопируй файлы
cd /root
mkdir splitbot && cd splitbot

# Скопируй все файлы сюда, затем:
npm install

# Создай .env
cp .env.example .env
nano .env
# Вставь BOT_TOKEN, ANTHROPIC_API_KEY

# Запуск через PM2
pm2 start src/index.js --name splitbot
pm2 save
```

## Использование

1. `/new` — начать новый чек
2. Отправь фото чека → бот распознаёт позиции
3. Подтверди список
4. Напиши кто что брал в свободной форме:
   - `Иван: бургер, пиво; Маша: салат`
   - `Иван взял бургер и пиво, Маша салат, десерт поровну, Иван заплатил`
5. Получи итог с долгами

## Команды

- `/new` — новый чек (с фото)
- `/manual` — ввести позиции вручную
- `/cancel` — сбросить сессию
